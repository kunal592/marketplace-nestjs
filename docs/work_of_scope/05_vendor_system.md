# Vendor System

## Vendor Registration

Users can apply to become vendors.

Vendor status:

PENDING
APPROVED
REJECTED

Admin must approve vendors before they can sell products.

---

# Vendor Profile

Vendor store includes:

store name
store slug
store logo
store description

---

# Vendor Dashboard Features

Vendors must be able to:

* add products
* edit products
* view orders
* track earnings
* request payouts
